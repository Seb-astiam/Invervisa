| Carpeta     | Propósito                                                                                             |
| ----------- | ----------------------------------------------------------------------------------------------------- |
| `core`      | Contiene lógica esencial y global, como servicios base, guards, interceptores de JWT, modelos         |
| `shared`    | Contiene componentes reutilizables y sin lógica propia del negocio (ej. botón personalizado, spinner) |
| `auth`      | Login, registro, almacenamiento de token, logout                                                      |
| `products`  | Listado, detalle, creación, edición (se adapta según el rol: admin o cliente)                         |
| `cart`      | Todo lo relacionado al carrito de compras                                                             |
| `orders`    | Creación de órdenes desde carrito, historial del usuario, control admin                               |
| `addresses` | Crear, actualizar y seleccionar direcciones para envío                                                |
| `admin`     | Panel administrativo (gestión de productos, usuarios, pedidos)                                        |
| `layout`    | Componentes visuales como navbar, sidebar, footer                                                     |
| `models`    | Contiene las interfaces (DTOs, tokens, usuarios, etc.) replicando las del backend                     |
